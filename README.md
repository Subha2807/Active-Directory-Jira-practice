# Active Directory & Jira Practice Project

**Status:** In progress

## Why I'm building this

I'm building hands-on experience with Active Directory (identity and access management) and Jira (ticket logging and tracking) — two tools widely used in IT support, service desk and security operations. This is a self-directed project to properly understand how they work in practice, rather than only reading about them.

## Goals

- Set up and configure a basic Active Directory environment (users, groups, organisational units, permissions)
- Practise common AD admin tasks, including account creation, password resets and access control changes
- Use Jira to log, track and update tickets as though handling real support requests
- Document each step clearly, including any issues faced and how I resolved them
